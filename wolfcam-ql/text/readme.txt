В эту папку можно закинуть любое фото и оно станет шейдером.
Фото может быть с альфа слоем (т.е с прозрачностью).
Формат фото - png, максимальное разрешение - 1500х1500 пикселей 
+ его пропорция должна быть равна (100x100, 200x200, 750x750, т.е КВАДРАТ).

Также имя должно соответствовать от d1 до d99.
Шейдер соответствует имени d1 до d99.

Настройка цвета в fx файле в таких шейдерах не поддерживается.
======================================================
In this folder you can throw any photo and it will become a shader.
The photo can be with an alpha layer (ie with transparency).
The format of the photo is png, the maximum resolution is 1500x1500 pixels
+ its proportion should be equal to (100x100, 200x200, 750x750, ie SQUARE).
Also the name must match from d1 to d99.
The shader corresponds to the name d1 to d99.

Color adjustment in the fx file in such shaders is not supported.
